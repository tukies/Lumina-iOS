# Lumina Website -> https://tukies.github.io/Lumina-iOS/

Marketing site for Lumina, built with React and Vite, deployed through GitHub Pages.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The GitHub Pages workflow in `.github/workflows/deploy.yml` deploys the site on pushes to `main`.
